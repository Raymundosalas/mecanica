# Videojuego 2D en Unity

Este proyecto consiste en el desarrollo de un videojuego 2D creado con el motor Unity.

## Mecánicas implementadas

- Movimiento del jugador
- Sistema de salto con físicas
- Recolección de objetos
- Sistema de puntuación
- Colisiones con enemigos

## Tecnologías utilizadas

- Unity
- Lenguaje C#
- Sistema de físicas Rigidbody2D
- Control de colisiones Collider2D

## Funcionamiento

El jugador puede desplazarse por el escenario utilizando las teclas de dirección y saltar con la barra espaciadora. Durante el recorrido puede recolectar objetos para obtener puntos. Si el jugador colisiona con un enemigo, la escena se reinicia.
